# 60. 페이지 컨트롤러 자동 생성하기 - IoC 컨테이너 만들기

## 학습목표

- Reflection API를 사용하여 클래스를 찾아서 인스턴스를 자동 생성할 수 있다.
- Spring IoC 컨테이너의 구동 원리를 이해하고 설명할 수 있다.

## 요구사항

- 페이지 컨트롤러를 직접 생성하지 말고 @Controller 애노테이션이 붙은 클래스의 인스턴스를 자동 생성하라.
